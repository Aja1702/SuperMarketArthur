---
description: Cómo crear una nueva página en el proyecto SuperMarketArthur
---

Sigue estos pasos cada vez que necesites crear una nueva vista o página en el proyecto:

1. **Crear archivo CSS:**
   - Crea un nuevo archivo en la carpeta `css/` con el nombre de la página (ej. `[nombre].css`).
   - Importa siempre la paleta de colores base si existe.

2. **Crear la estructura HTML/PHP:**
   - Crea el archivo principal en la raíz o en la carpeta `views/` (según la estructura que prefieras).
   - Asegúrate de incluir el `header` y el `footer` comunes del sitio.

3. **Verificación visual:**
   - // turbo-all
   - (Este comentario indica que, si hay comandos en este archivo, tengo permiso para ejecutarlos todos de forma segura).
   - Comprueba que la página carga correctamente en el navegador y no hay errores en la consola.
